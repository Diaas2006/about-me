Reverse-Shell

Uma tentativa de realizar uma reverse shell em um ambiente controlado. No entanto, ocorreu um erro, pois a biblioteca DigiKeyboard, utilizada em conjunto com o Digispark ATtiny85, estava digitando o código da shell utilizando o layout de teclado US, que difere dos layouts PT-BR/PTB2, uma vez que determinados caracteres estão posicionados em locais diferentes.
